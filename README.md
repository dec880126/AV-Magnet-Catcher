## Feature
 - 可提取 sehuatang 之相關資訊
 - 可提取 草榴社區 之相關資訊
 - 整合以上兩個網站之抓取系統
 - 整合 Auto SHT
 - 串接 Synology Web API
 - 可自定義抓取影片風格(透過關鍵字篩選實現)
 - 未來: 圖形化使用者介面(GUI)
